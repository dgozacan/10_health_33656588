This is a web-based health and fitness tracking application built using Node.js, Express, and MySQL. 
The app allows users to log their workouts, view their workout history, and plan what body part they want to train next. 
It has been developed as part of a Dynamic Web Applications coursework assignment.

Technologies Used:
Node.js
Express
MySQL
EJS
bcrypt
express-session
mysql2

Features:
User login with session-based authentication
Add workouts with details such as: workout name, body part trained, duration, notes, planned next body part
View the most recent workout and planned next workout on the home page
Search workout history using keywords
Data stored in a MySQL database
